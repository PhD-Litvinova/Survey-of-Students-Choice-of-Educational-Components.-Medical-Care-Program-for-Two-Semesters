# Students’ Choice of Educational Components

## Overview

This dashboard presents the results of a survey on students’ selection of elective educational components for the **Medical Care** program over two semesters. The analysis highlights the popularity of practical and theory-based courses.

## Steps to Create the Dashboard

1. **Data Collection**

   * Received raw data in a Word document from the college.
   * Extracted all relevant fields (course names, program, semester, number of students) into a structured table.

2. **Data Cleaning and Normalization**

   * Transferred data to Google Sheets.
   * Checked for duplicates, missing values, and consistency in course names.
   * Normalized course titles to ensure uniformity across semesters.

3. **Data Analysis**

   * Calculated total number of students per course and per semester.
   * Converted raw numbers into percentages for comparative analysis.
   * Identified most popular courses for each program and semester.

4. **Visualization**

   * Imported cleaned data into Tableau.
   * Built histograms and bar charts for course selection per semester and overall.
   * Added filters to enable dynamic exploration by semester and program.

5. **Insights and Conclusions**

   * Summarized findings in a concise report highlighting the preference for practical courses.
   * Prepared a presentation in PowerPoint with key visualizations and insights.
